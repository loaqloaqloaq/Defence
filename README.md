- 制作環境： UNITY
- 実行環境： Windows

## ゲーム
- ゲーム名：Robot Harzard
- ゲーム内容：ロボットから基地を守る。敵拠点壊すか、時間制限まで守りきると勝ち
- 人数：１

## 操作

キーボード
- 移動：W/A/S/D
- ジャンプ： SPACE
- 攻撃： 左クリック
- 走る: 左SHIFT
- マップ: M
- 武器切り替え: 1/2/3/4
- 使用: F
